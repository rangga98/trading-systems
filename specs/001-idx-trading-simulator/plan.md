## Summary

Build a trading simulation web application for the Indonesian Stock Exchange (IDX). The system enables traders to import historical OHLCV data for IDX tickers via yfinance, visualize data as interactive candlestick charts, and run backtests with configurable money management parameters (initial capital, position sizing, stop loss, take profit). Results include P&L, equity curves, win rate, and max drawdown analysis.

**Technical Approach**: FastAPI backend with async SQLAlchemy ORM, PostgreSQL for persistence, React frontend with lightweight-charts for candlestick visualization. All backtest computation runs server-side for determinism and data integrity.

## Technical Context

**Language/Version**: Python 3.12+, TypeScript/React 18+  
**Primary Dependencies**: FastAPI (async), SQLAlchemy 2.0, yfinance, pandas, lightweight-charts, TanStack Query  
**Storage**: PostgreSQL with asyncpg driver  
**Testing**: pytest (backend), Vitest (frontend)  
**Target Platform**: Web browser (Chrome, Firefox, Safari, Edge), Docker Compose local dev  
**Project Type**: Web application (backend + frontend SPA)  
**Performance Goals**: Import 5 years data <30s, chart render 1000 points <2s, backtest 1250 bars <5s  
**Constraints**: All monetary values as NUMERIC(19,4), WIB timezone (UTC+7), single-user MVP  
**Scale/Scope**: Single deployment, 600+ IDX tickers, daily granularity only
